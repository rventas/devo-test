El ejercicio 2 se compone de una aplicación desarrollada en Flask, que lee de tres fuentes de datos y los  unifica. 
Con el resultado se muestran dos gráficas con highcharts, una de líneas con una línea por cada una de las categorías reflejadas
en los datos y un pie chart con la suma total por categoría. 
La aplicación se encuentra en el archivo devo-test-ejercicio2. La ruta para acceder a la página que muestra las gráficas es 
localhost:9999
